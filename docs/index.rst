Welcome to Rangelands Decision Support Tool documentation
=========================================================

The Rangelands Decision Support Tool has been developed by SERVIR Eastern and Southern Africa (SERVIR ESA). SERVIR-ESA is a joint initiative between NASA (National Aeronautical Space Agency) and USAID with Regional Center for Mapping of Resources for Development (RCMRD) as the implementing agency. How To Use The Documentation
============================

The documentation is geared toward two distinct types of users:

 #. **Users**: Are people who log into the website and use its functionality.
 #. **Developers**:  Are people who write code to add functionality.


The documentation is divided into three sections:

 #. :doc:`Tutorials: <help/index>` Step-by-step instructions in workshop format that help a user to accomplish a set of tasks.
 #. :doc:`Reference: <developer/index>` Architecture, component information, API descriptions etc.
